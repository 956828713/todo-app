# 待办事项应用

一个简洁的单文件待办事项应用，支持明暗模式和中英文切换。

## 功能特性

- ✅ **增删改查**：添加、删除、标记完成待办事项
- 🌗 **主题切换**：明暗模式一键切换，自动保存偏好
- 🌐 **多语言**：中英文界面实时切换
- 📱 **响应式设计**：适配桌面和移动设备
- 💾 **本地存储**：数据保存在浏览器 localStorage 中
- 🔍 **筛选功能**：查看全部、未完成、已完成事项
- 🧹 **清理功能**：一键清除所有已完成事项

## 使用方法

1. 直接双击 `index.html` 在浏览器中打开
2. 无需服务器、无需安装依赖

### 基本操作

- **添加待办**：在输入框中输入内容，点击"添加"按钮或按 Enter 键
- **标记完成**：点击待办事项前的复选框
- **删除待办**：点击待办事项右侧的 ✕ 按钮
- **筛选查看**：使用上方的"全部"、"未完成"、"已完成"按钮
- **清除已完成**：点击底部的"清除已完成"按钮
- **切换主题**：点击右上角的主题切换按钮
- **切换语言**：点击右上角的语言切换按钮

## 技术栈

- **HTML5**：语义化标签，现代 API
- **CSS3**：CSS 变量主题系统，Flexbox 布局
- **原生 JavaScript**：无框架，DOM 操作，localStorage API
- **无构建工具**：单文件应用，零依赖

## 数据模型

每条待办事项存储为 JSON 对象：

```json
{
  "id": 1649945678901,
  "text": "学习 JavaScript",
  "done": false
}
```

数据保存在浏览器的 `localStorage` 中，键名为 `todos`。
